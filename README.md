# Busy Bees Motel

Warm, animated boutique motel landing page.

**Stack:** Vite · React · TypeScript · Tailwind CSS · Framer Motion · Bun

## Sections
- Hero with animated bee mascot
- Rooms
- Amenities
- Gallery
- About
- Testimonials
- Booking form → WhatsApp
- Footer

## Run locally

```bash
bun install
bun dev
```

## Build

```bash
bun run build
```
